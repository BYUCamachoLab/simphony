=====================
Simphony Integrations
=====================

Simphony integrates well with several other programs. Additionally, Simphony
has modules that allow other software to integrate well into Simphony. 
These are described here.

If you have written an integration for Simphony that is publically available,
please let us know so we can add it to this list!
